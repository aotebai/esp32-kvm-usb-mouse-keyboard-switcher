# esp32-kvm-usb-mouse-keyboard-switcher
A dual‑device KVM switch using ESP32‑S3 and CH9350 enables seamless keyboard and mouse sharing between two PCs. An onboard RGB‑LED shows status. Driver‑free, it allows users to flash precompiled firmware in one click. A redundant power‑supply design ensures stable operation even when only one device is powered on.
